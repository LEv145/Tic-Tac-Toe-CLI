# Tic Tac Toe (CLI)
Простое ООП, на Java, может поможет новичкам

Компиляция:
```
javac -sourcepath ./src -d bin C:\JAVA\KN\src\com\company\Main.java
```
(Ps: Вместо C:\JAVA\KN\src\com\company\Main.java - свой путь)

Запуск:
```
java -classpath ./bin com.company.Main
```
